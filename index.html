Say Hello to Vagrantpress

Vagrantpress is a simple development enviroment for creating and modifying WordPress sites. Using Vagrantpress is meant to be easy - explode a zip (or .tar.gz) file into a directory and type 

$ vagrant up

Once vagrant builds the environment, you'll have a clean development environment for creating your own WordPress modules and themes.

What Do I Need to Get Started?

You'll need a working copy of the Vagrant virtualization environment, as well as a working copy of VirtualBox.

In the short term, you'll also need to provide your own 'box' until I get a recent copy of an Ubuntu box up and running. (Or at least one that downloads in reasonable times on reasonable bandwidths.) In the meantime, you can edit the Vagrantfile to use your own box that supports apt-get and git. The included puppet script should build the rest of your environment with those basic commands.

What Does VagrantPress include?

VagrantPress will build a very simple LAMP stack, with an Apache 2 install, a PHP 5 install and a MySQL install. At present, these systems (and the puppet script) work against an Ubuntu 12.04 LTS server box. The puppet scripts will also clone the WordPress repository from GitHub, which should allow you to use many different versions of WordPress. (The puppet script defaults to the latest release version of 3.5.)

Authors and Contributors

Vagrantpress is a personal project of @chad-thompson who was looking for a simpler way to stand up multiple Wordpress environments. This project is one of the first 'real' Vagrant environments (and puppet scripts) that I've put together, but I hope that someone else finds this as useful as I do.
